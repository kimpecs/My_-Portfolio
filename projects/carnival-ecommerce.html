Project Overview

A complete event management solution for carnival organizers featuring ticket sales, inventory management, and attendee analytics.

Key Features

  • Real-time ticket inventory
  • Stripe payment integration
  • Event management dashboard
  • Attendee analytics

Full Tech Stack

  • Frontend: Vue.js, Vuetify, Chart.js
  • Backend: Flask, JWT Auth
  • Database: PostgreSQL
  • DevOps: Docker, CI/CD
