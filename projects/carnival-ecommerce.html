Tools: Vue.js, Flask, PostgreSQL, Stripe API, Docker

A full-stack event management and ticket sales platform designed for carnival and festival organizers with real-time inventory management and secure payment processing.

Key Features

  • Event creation and management dashboard
  • Real-time ticket inventory system
  • Secure payment processing with Stripe
  • Attendee management and analytics
  • Mobile-responsive design

Technical Architecture

The application follows a modern microservices architecture:

  • Frontend: Vue.js with Vuetify for UI components
  • Backend: Flask REST API with JWT authentication
  • Database: PostgreSQL with Redis caching
  • DevOps: Docker containers with CI/CD pipeline
